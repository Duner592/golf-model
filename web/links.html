Useful Links and Tools

External resources and tools for golf modeling, data analysis, and betting insights.

External Links

  • DataGolf API - Primary data source for historical rounds, stats, and odds.
  • PGA Tour Official Site - Leaderboards, player profiles, and event schedules.
  • DP World Tour (European Tour) - Stats, results, and European event info.
  • Official World Golf Ranking - Player rankings and historical data.
  • Golf Betting System - Useful Insight, prediction models, data and FRL stats.
  • Odds Checker - Check latest golf odds
  • Sporting Life - Golf - Golf Articles and tips
  • Racing Post - Golf - Golf Articles and tips
  • Golf Odds - Odds Archive

Tools

  • GitHub golf-model repo - Duner592 - golf-model git repository.

  • Google Colab - Free Jupyter notebooks for running Python scripts.
  • Pandas Documentation - Data manipulation library reference.
  • Tableau Public - Free data visualization for golf stats analysis.
  • Desmos Graphing Calculator - Plotting trends like strokes gained.
  • Python Official Site - Download and learn Python for scripting.
  • Jupyter Notebooks - Interactive coding environment.
  • Anaconda - Data science platform with Python packages.
  • SQLite - Lightweight database for local data storage.
  • Docker - Containerization for reproducible environments.
